Developer Pricing

TringMe APIs are available for FREE to the developer community to integrate in all kinds of applications, including commercial applications.

Free APIs & Source Code

Not only APIs, but complete source code including fully working sample applications, helper files which make it easier to invoke the APIs in specific languages are available for developers to use and change as they wish.

Free Support

In addition to sample source files,API documentation, FAQ etc., TringMe also provides free support to the developer community to ensure that they can integrate TringMe APIs in their application and get going quickly. There are no charges for using TringMe support either.
